# Stemgen GUI Icons

This directory contains app icons for different platforms.

## Required Icons

- `icon.png` - Base icon (1024x1024 recommended)
- `icon.ico` - Windows icon
- `icon.icns` - macOS icon
- `32x32.png` - Small Windows icon
- `128x128.png` - Medium Windows icon
- `128x128@2x.png` - Retina macOS icon

## Generating Icons

You can use the Tauri icon generator:

```bash
npm run tauri icon
```

Or use a tool like:
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [Image2Icon](http://www.img2icnsapp.com/)
- [CloudConvert](https://cloudconvert.com/)

## SVG Source

An SVG source file is recommended for best quality at all sizes.
